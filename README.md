# Roadmap Học Mạng Máy Tính

## Giai đoạn 1: Nền tảng cơ bản (1-2 tháng)

### Khái niệm cơ bản
- **Mạng máy tính là gì?**
  - Định nghĩa và lợi ích
  - Các loại mạng: LAN, WAN, MAN
  - Topology mạng: Star, Bus, Ring, Mesh

- **Mô hình OSI và TCP/IP**
  - 7 tầng mô hình OSI
  - 4 tầng mô hình TCP/IP
  - Chức năng từng tầng
  - Encapsulation và Decapsulation

### Phần cứng mạng
- **Thiết bị mạng cơ bản**
  - Hub, Switch, Router
  - Modem, Access Point
  - Network Interface Card (NIC)
  - Cáp mạng: UTP, STP, Fiber optic

## Giai đoạn 2: Giao thức và địa chỉ IP (2-3 tháng)

### Địa chỉ IP
- **IPv4**
  - Cấu trúc địa chỉ IPv4
  - Subnet mask và CIDR
  - Phân loại địa chỉ IP: Public, Private
  - Subnetting và VLSM

- **IPv6**
  - Cấu trúc địa chỉ IPv6
  - So sánh IPv4 vs IPv6
  - Transition mechanisms

### Giao thức quan trọng
- **TCP vs UDP**
  - Đặc điểm và so sánh
  - Khi nào dùng TCP, khi nào dùng UDP
  - Port numbers

- **DHCP, DNS**
  - Cách hoạt động DHCP
  - Hệ thống phân giải tên miền DNS
  - Cấu hình cơ bản

## Giai đoạn 3: Switching và VLANs (1-2 tháng)

### Switching
- **Ethernet Switching**
  - MAC address table
  - Collision domain vs Broadcast domain
  - Spanning Tree Protocol (STP)

- **VLANs**
  - Virtual LAN là gì?
  - Trunk và Access ports
  - VLAN Trunking Protocol (VTP)
  - Inter-VLAN routing

## Giai đoạn 4: Routing (2-3 tháng)

### Routing cơ bản
- **Static Routing**
  - Cấu hình route tĩnh
  - Default route
  - Administrative distance

- **Dynamic Routing**
  - Distance Vector vs Link State
  - RIP, OSPF, EIGRP
  - BGP cơ bản

### Routing nâng cao
- **OSPF**
  - Single area và Multi-area
  - LSA types
  - Neighbor adjacency

- **BGP**
  - eBGP vs iBGP
  - Path attributes
  - Route selection process

## Giai đoạn 5: Wireless và Bảo mật (2-3 tháng)

### Wireless Networking
- **Wi-Fi Standards**
  - 802.11 a/b/g/n/ac/ax
  - Wireless security: WEP, WPA, WPA2, WPA3
  - Wireless controllers

### Network Security
- **Firewall**
  - Types of firewalls
  - ACLs (Access Control Lists)
  - NAT và PAT

- **VPN**
  - Site-to-site VPN
  - Remote access VPN
  - IPSec, SSL VPN

## Giai đoạn 6: Monitoring và Troubleshooting (1-2 tháng)

### Network Monitoring
- **SNMP**
  - MIB và OID
  - Network monitoring tools
  - Performance metrics

### Troubleshooting
- **Network Troubleshooting**
  - Ping, Traceroute, Nslookup
  - Wireshark packet analysis
  - Common network issues

## Giai đoạn 7: Chuyên sâu (3-6 tháng)

### Network Design
- **Enterprise Network Design**
  - Hierarchical design model
  - Redundancy và High Availability
  - QoS (Quality of Service)

### Automation
- **Network Automation**
  - Python for networking
  - Ansible, Puppet
  - SDN (Software Defined Networking)

## Tài liệu và Lab thực hành

### Sách khuyến nghị
- "Network+ Guide to Networks" - Tamara Dean
- "CCNA 200-301 Official Cert Guide" - Wendell Odom
- "TCP/IP Illustrated" - W. Richard Stevens

### Lab Environment
- **Cisco Packet Tracer** (miễn phí)
- **GNS3** (mô phỏng mạng)
- **EVE-NG** (emulation platform)
- **Virtual Box/VMware** cho lab

### Certification Path
1. **CompTIA Network+** (entry-level)
2. **Cisco CCNA** (associate level)
3. **Cisco CCNP** (professional level)
4. **Cisco CCIE** (expert level)

## Timeline và Lời khuyên

### Thời gian học
- **Tổng thời gian**: 12-18 tháng (học part-time)
- **Mỗi tuần**: 10-15 giờ học và thực hành
- **Tỷ lệ lý thuyết/thực hành**: 40/60

### Lời khuyên
- Luôn kết hợp lý thuyết với thực hành lab
- Tham gia cộng đồng networking trên các forum
- Thực hiện projects thực tế
- Chuẩn bị cho certification để có mục tiêu cụ thể
- Tập trung vào troubleshooting skills

### Bước tiếp theo
Sau khi hoàn thành roadmap này, bạn có thể chuyên sâu vào:
- Cloud Networking (AWS, Azure, GCP)
- Network Security (Cyber Security)
- Data Center Networking
- Service Provider Networking
- Network Automation và DevOps
